プライバシーポリシー

就活Dash! — Chrome拡張機能

1. 収集する情報

就活Dash!（以下「本拡張機能」）は、以下の情報をお客様のデバイス上にのみ保存します。

プロフィール情報
氏名（漢字・フリガナ）、性別、生年月日、住所、郵便番号、電話番号、メールアドレス、学校名、学部・学科、入学・卒業年月、ゼミ・クラブ

就活管理情報
企業名、マイページURL、選考状況、締切日、ESの設問と回答、面接メモ、タグ・メモ

2. 情報の保存場所

すべてのデータはお客様のブラウザ内（chrome.storage.local および chrome.storage.sync）にのみ保存されます。外部サーバーへの送信は一切行いません。

3. 外部サービスとの通信

本拡張機能は以下の外部サービスとのみ通信します。個人情報（氏名・住所・メール等）を外部に送信することはありません。

サービス 用途 送信するデータ
Google Favicon API
(t2.gstatic.com)		 企業ロゴの表示 企業ドメイン名
Google 検索
(google.co.jp)		 企業ホームページの自動取得 企業名（検索クエリ）
Google カレンダー / Outlook 締切日の登録（ユーザー操作時のみ） 企業名・日時

4. 第三者への提供

収集した情報を第三者に販売・提供・開示することはありません。

5. データの削除

拡張機能をChromeから削除すると、保存されたすべてのデータが削除されます。ダッシュボードの「設定」ページからもデータを手動で削除できます。

6. お問い合わせ

本ポリシーに関するお問い合わせは、Chrome ウェブストアのサポートページよりご連絡ください。
